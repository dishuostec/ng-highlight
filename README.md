ng-highlight v0.1
============

This is a Angularjs version of [jQuery Highlight Plugin](http://bartaz.github.io/sandbox.js/jquery.highlight.html).

It can be used to highlight pieces of text on a page.

Dependence
--------------
Angularjs 1.2+
